PHP Group Project

This was a class project I did in PHP. I created most of the website or had a helping hand in the website besides the contact page and the calorie page. The site gives you a list of ingredients from your pantry to see if you can make recipies. It then takes the used items and deletes them from the pantry. It has an admin page that lets you enter more ingredients into the database. It also lets you update or delete the current ingredients from the pantry. The database a used is from the relational databases from Amazon Web Services. I used MySQL Workbench to create some of the tables and enter in ingredients, user names, and passwords. The login, insert, update pages have server side validation used to check the fields. I have disabled new user sign ins so if you wish to see the admin page please contact me with a request and I will give you access.
